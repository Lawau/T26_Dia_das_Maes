# 🍫 Reserva de Docinhos — Dia das Mães

Site de reservas de docinhos artesanais para o Dia das Mães.

## Como usar

Abra o `index.html` direto no navegador — não precisa de servidor.

## Funcionalidades

- Formulário de reserva em 4 etapas (dados → caixas → sabores → resumo)
- 4 tipos de caixa: 4, 6, 6+rosa e 9 doces
- 8 sabores: Brigadeiro, Beijinho, Ninho, Bicho de Pé + 4 Trufas
- Mistura de sabores por caixa
- Geração de código de reserva único
- Download do comprovante em JPEG
- Painel administrativo com senha para ver e exportar reservas (CSV)

## Painel Admin

Clique em "Área Administrativa" no rodapé e use a senha configurada no código (`ADMIN_SENHA`).

## Publicação

Hospedado via **GitHub Pages**

## Observação sobre dados

As reservas ficam salvas pelo JSONBin.
